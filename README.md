
# DocuSpace Frontend

Frontend for DocuSpace — an AI-powered document workspace platform.

---

## 🧠 Overview

This React-based frontend provides the user interface for:

- Document management
- Workspace navigation
- Search functionality
- Integration with backend APIs

---

## 🛠️ Tech Stack

- React (Vite + TypeScript)
- React Query
- Axios
- React Router

---

## 🔗 Backend Repository

Backend service:  
👉 https://github.com/nikhilpeshwe-rgb/docuspace-backend

---

## ⚙️ Features

- Workspace and document navigation  
- Document creation and editing  
- Search interface (API-driven)  
- Integration with async backend processing  

---

## 🚀 Running Locally

npm install
npm run dev
